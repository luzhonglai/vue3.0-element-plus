# vue-multiple-pages

> 项目模版使用 Vue3/Ts, 实现多页面模块开发模式。

## 项目亮点

- 自动化发布部署、gitlab-ci 代码测试
- 代码风格统一、eslint+ts+prettier
- mock 数据模拟、随机生成数据表
- 接口 log 窗口调试，

### 目录

```base
├── mock
│   └── modules
├── public
├── src
│   ├── api
│   ├── assets
│   ├── common
│   ├── layout ----- 公告布局
│   ├── plugins ---- 模版插件
│   ├── router  ----
│   ├── store
│   ├── styles
│   ├── types
│   ├── utils
│   └── views
└── tests
    └── unit
```
